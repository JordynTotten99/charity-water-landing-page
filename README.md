# charity-water-landing-page
Responsive charity landing page built with HTML and CSS.
This project is a responsive nonprofit landing page built using HTML and CSS. The design was initially created in canva and then translated into code using AI.
## Project Goals 
-Recreate a Canva mockup using HTML and CSS
-Implement responsive design
-Add hover effects and CSS transitions
-Deploy using GitHub Pages
## Technologies Used 
-HTML5
-CSS3
-GitHub Codespaces
-GitHub Pages
